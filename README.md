# django-invoices
Create invoices using django

## Pre-requisites (linux)
* wkhtmltopdf 0.12.6 (with patched qt)
* sudo apt install wkhtmltopdf

## Usage
1) python manage.py migrate
2) python manage.py runserver 
3) Go to http:localhost:8000
